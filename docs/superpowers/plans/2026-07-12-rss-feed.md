# RSS Feed Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Make Hugo's existing summary RSS feed discoverable and limit it to the 20 newest posts.

**Architecture:** Retain Hugo's built-in RSS output and configure its item limit globally. Resolve the home page's RSS output from shared templates so pages rendered through the theme advertise the same canonical feed URL.

**Tech Stack:** Hugo 0.163.x, Go templates, YAML, HTML, RSS 2.0

---

### Task 1: Establish the current feed behavior

**Files:**
- Inspect: `public/index.xml`

- [ ] **Step 1: Build the unmodified site**

Run: `hugo`

Expected: The build succeeds and creates `public/index.xml`.

- [ ] **Step 2: Confirm the desired assertions initially fail**

Run:

```bash
test "$(rg -c '<item>' public/index.xml)" -le 20
rg 'rel="alternate"[^>]+application/rss\+xml' public/index.html
rg '<a[^>]+href="/index.xml"[^>]*>RSS</a>' public/index.html
```

Expected: At least the item-limit assertion and discovery-link assertions fail before implementation.

### Task 2: Limit the built-in RSS feed

**Files:**
- Modify: `config.yaml`

- [ ] **Step 1: Configure the RSS limit**

Add:

```yaml
services:
  rss:
    limit: 20
```

- [ ] **Step 2: Build and verify the item limit**

Run:

```bash
hugo
test "$(rg -c '<item>' public/index.xml)" -eq 20
```

Expected: Hugo succeeds and the assertion exits successfully.

### Task 3: Add feed discovery links

**Files:**
- Modify: `themes/webcodr/layouts/partials/head.html`
- Modify: `themes/webcodr/layouts/partials/footer.html`

- [ ] **Step 1: Add RSS autodiscovery metadata**

Add inside `<head>`:

```go-html-template
{{ with (.Site.Home.OutputFormats.Get "RSS") }}
  <link rel="alternate" type="{{ .MediaType.Type }}" href="{{ .RelPermalink }}" title="{{ $.Site.Title }}" />
{{ end }}
```

- [ ] **Step 2: Add the footer link**

Add to `.footer-menu`:

```go-html-template
{{ with (.Site.Home.OutputFormats.Get "RSS") }}
  <li class="footer-item">
    <a href="{{ .RelPermalink }}" class="footer-link">RSS</a>
  </li>
{{ end }}
```

- [ ] **Step 3: Build and verify generated HTML**

Run:

```bash
hugo
xmllint --html --xpath 'string(//link[@rel="alternate"]/@type)' public/index.html | rg '^application/rss\+xml$'
rg '<a href="/index.xml" class="footer-link">RSS</a>' public/index.html
xmllint --html --xpath 'string(//link[@rel="alternate"]/@href)' public/about/index.html | rg '^/index.xml$'
```

Expected: Hugo succeeds and all three searches match.

### Task 4: Validate the generated RSS document

**Files:**
- Verify: `public/index.xml`

- [ ] **Step 1: Parse and inspect the feed**

Run:

```bash
xmllint --noout public/index.xml
test "$(rg -c '<item>' public/index.xml)" -eq 20
rg '<description>' public/index.xml
```

Expected: XML parsing succeeds, exactly 20 items exist, and summary descriptions remain present.
